# DOCBOTICS COMPUTER VISION PYTHON EXAMPLES 
***
## HOW TO RUN EXAMPLES 
To run the following examples you will need to have the following: 
#### Software:  
Python 3.11 or Greater 
OpenCV 4.13.0 or Greater
#### Hardware: 
Webcam

### LINUX / MACOS  
*** 
Run the following commands in your terminal or command prompt if you are on windows to get the following dependencies
	pip3 install python-opencv  

Navigate to the directory that has the example code and  run the following command. Be sure to replace the filename with the filename of the example that you want to run. 
python3 ball_tracking.py
